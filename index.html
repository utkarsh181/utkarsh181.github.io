RSS feeds

A web “feed”, also known as “RSS feed” or “Atom feed”, is a stream of data that gets updated every time something new is added to the webpage it tracks. For example, a blog’s feed will contain all newly published articles.

The usefulness of this tool is that it allows specialised programs or applications-feed readers-to automatically check multiple websites for updates, based on the list of feed subscriptions.

To subscribe, the user will typically have to copy the feed’s link into their reader’s appropriate place.
